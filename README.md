# **Vote Plugin**

This plugin gives some rewards to a player who voted in the server.

## How to add rewards?
First of all you need understand that the rewards are given by a command (like `/time set day `sended by the console. When a command need the player nick just add "USER" (/kit tools USER).
`commands: ["kit tools USER"]`
